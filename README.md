📌 Project Overview

This application is a web-based tool designed to manage and display personalized interview schedules for students.

Users first enter their first and last name on a login page. The app then retrieves their information from a data source (converted from an Excel file into JSON) and displays a customized page with key details such as:

Interview time
Assigned room
Accompanying staff member
